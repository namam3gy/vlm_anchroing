\begin{abstract}
\todoNote{목표 분량 $\sim$180 단어. 아래 5개 beat을 한 문단으로 압축.}
\textbf{Problem.} VLM에게 질문과 무관한 이미지를 함께 보여줄 때, 그 이미지에 숫자가 그려져 있다면 답이 그 숫자 쪽으로 끌리는가 -- \textit{cross-modal numerical anchoring}.
\textbf{Finding 1 (현상).} 6개 open-weight VLM $\times$ 5 dataset에서 약 10--40\,\% 의 응답이 anchor 숫자 쪽으로 \textit{점진적으로} 끌리고, 그 중 일부 (1.7--15.7\,\%) 는 anchor 를 그대로 베끼기까지 한다. 끌리는 정도는 base 답에 대한 모델 confidence 가 낮을수록 더 크다.
\textbf{Finding 2 (causal gate).} 같은 anchor 이미지에서 숫자 픽셀만 가린 \textit{masked} 짝을 만들어 비교하면 위 끌림이 일반 distractor 수준으로 사라진다 -- 즉 anchoring 은 \textit{(보이는 digit)} $\times$ \textit{(모델 uncertainty)} 두 조건의 conjunction.
\textbf{Method.} 본 논문은 anchored 와 masked 짝의 \textit{paired contrast} 를 신호로 삼아 모델 내부의 anchoring representation 을 추정하고, inference 시 이를 제거하는 mitigation 을 제안한다.
\textbf{Result.} 5 dataset 위에서 anchoring 효과 감소 + anchor 가 있는 경우와 없는 경우 모두 정확도 동시 상승, 6 held-out capability benchmark 평균 보존 ($+0.41$\,pp).
\textbf{Implication.} Mechanism 분석은 anchoring signal 이 단일 layer 가 아니라 모델 후반부 여러 layer 에 분산되어 존재함을 보여, vision-modality bias 가 LM 후반부에 \textit{분리 및 제거 가능한 representation 단위} 로 존재해 retraining 이나 prompt-level 방어 없이 inference 시 직접 개입할 수 있음을 시사한다.
\end{abstract}

\section{Introduction}

\subsection{동기 (Motivation)}
\begin{itemize}
  \item \textbf{Hook (model-behavior question).} Multi-image prompt 가 RAG / multi-screenshot Q\&A / VLM-as-judge 등에서 일반화되는 가운데 -- VLM 은 질문과 무관한 시각 입력을 \textit{무시} 하는가, 아니면 그 정보가 답을 \textit{흔드는가}?
  \item \textbf{Phenomenon 명명.} \textit{Cross-modal numerical anchoring} -- 무관한 이미지의 digit pixel이 모델 답을 끌어당긴다.
  \item \textbf{Cognitive grounding.} Anchoring 은 인간 [Tversky and Kahneman, 1974] 과 텍스트 LLM [Jones and Steinhardt, 2022; Echterhoff et al., 2024] 에서 잘 정립된 인지 편향. 그러나 \textit{시각 modality} 위에서 -- 즉 visual cue 가 모델의 답을 어떻게 끌어당기는지에 대해서는 체계적 평가가 부재. 위 deployment 시나리오 (RAG retrieval, multi-image Q\&A, VLM-as-judge) 에서 무관한 시각 입력 노출이 자연스럽게 발생할 수 있어, 이 빈 칸은 cognitive 호기심을 넘어선 deployment relevance 를 가진다.
\end{itemize}

\subsection{본 논문의 발견 (Findings)}
\todoNote{Abstract 의 Finding 1 / Finding 2 를 본문 톤으로 한 단계 풀어서 서술. 4-condition 자극 (b: target only / a: + anchor / m: + masked anchor / d: + neutral distractor) 으로 6 open-weight VLM $\times$ 5 dataset 측정.}
\begin{itemize}
  \item \textbf{F1 (graded pull).} 약 10--40\,\% 의 응답이 anchor 쪽으로 점진적으로 끌리고, 그 중 일부 (1.7--15.7\,\%) 는 anchor 를 그대로 베낀다 -- 효과의 질량은 \textit{literal copy} 가 아닌 \textit{점진적 이동} 에 있다.
  \item \textbf{F2 (uncertainty modulation).} 끌리는 정도는 base 답에 대한 모델 confidence 가 낮을수록 크다 (5 dataset $\times$ 6 model L1 6-bin 위 단조 monotonic gradient).
  \item \textbf{F3 (digit-pixel causal gate).} Anchor 이미지에서 digit pixel 만 가린 \textit{masked} 짝과 비교하면 끌림이 일반 distractor 수준으로 사라진다 -- \textit{digit pixel} $\times$ \textit{uncertainty} 두 조건의 conjunction.
  \item \textbf{F4 (mechanism).} Anchoring 의 internal representation 은 LM 후반부 layer 에 형성된다 -- 이 위치는 본 논문 mitigation 의 작용 site 가 된다.
\end{itemize}

\subsection{접근과 결과 (Approach \& Result)}
\begin{itemize}
  \item \textit{(a $-$ m) paired contrast} 를 calibration 신호로 활용 $\rightarrow$ 모델 내부의 anchoring representation 을 추정 $\rightarrow$ inference 시 projection 으로 제거.
  \item Calibration 은 별도 phase 에서 한 차례만 수행되고, 거기서 얻은 projection 은 이후 모든 inference 에 일괄 적용된다. 배포 단계부터는 입력 이미지에 anchor 가 포함되어 있는지 여부와 무관하게 동일한 projection 이 작동하므로, runtime 에 anchor 를 탐지하거나 라벨링할 필요가 없다 -- 그대로 deployable 하다.
  \item 이 mitigation 을 적용하면 5 dataset 위에서 anchoring 효과가 감소하며, 동시에 anchoring 과 \textit{무관한} 6 개 general capability benchmark (held-out) 의 성능도 손상되지 않는다 (평균 변화 $+0.41$\,pp) -- 즉 mitigation 이 모델의 일반 능력을 해치지 않는다.
\end{itemize}

\subsection{기여 (Contributions)}
\begin{itemize}
  \item \textbf{C1 (Phenomenon).} Cross-modal numerical anchoring 을 5 dataset $\times$ 6 open-weight VLM 위에서 정량 보고 -- 약 10--40\,\% 의 응답이 무관한 이미지의 digit 으로 끌리며 (일부는 그대로 베끼고), 끌림은 모델 confidence 에 비례해 graded 되며, anchor 이미지에서 digit pixel 만 가리면 효과가 사라짐. 행동 + causal gate 동시 검증.
  \item \textbf{C2 (Mitigation).} \textit{(a $-$ m) paired contrast} 를 calibration 신호로 활용해 LM 후반부의 anchoring representation 을 한 차례 추정하고, inference 시 모든 입력에 projection 으로 일괄 적용. anchor label 이나 runtime 탐지 불필요. 5 dataset 에서 anchoring 효과 감소 + 6 일반 capability benchmark 평균 $+0.41$\,pp 보존.
  \item \textbf{C3 (Mechanism evidence).} Mechanism 분석으로 (i) anchoring representation 이 LM 후반부 layer 에 위치하고, (ii) 그 layer 안에서도 single direction 으로 reduce 되지 않음을 보여 -- \S6 mitigation 의 \textit{작용 site} 와 \textit{multi-direction subspace} 설계 모두에 근거를 제공.
\end{itemize}

\subsection{Prompt template}
모든 6개 모델, 모든 dataset, 모든 4-condition 에 대해 \textit{동일한} system + user template 사용 (모델별 변동 없음). 이미지 슬롯 개수만 condition 에 따라 1 (b) / 2 (a, m, d) 로 변동.

\paragraph{System.}
\begin{verbatim}
You are a visual question answering
system.
Return valid JSON only in the form
{"result": <number>}.
Use a numeric JSON value for <number>,
not a string.
Do not output any other keys, words,
explanation, or markdown.
If uncertain, still output the single
most likely number in that JSON format.
\end{verbatim}

\paragraph{User template.}
\begin{verbatim}
Answer the question using the provided
image(s).
Return JSON only in the form
{"result": <number>}.
Question: {question}
\end{verbatim}

Sampling: \texttt{temperature=0.0}, \texttt{top\_p=1.0}, \texttt{max\_new\_tokens=16}.

  \label{tab:anchor-inventory}

\paragraph{Per-question stratification.} For each question, distance $|a - gt|$ between GT and every candidate is computed. Within each distance stratum, one candidate is sampled via RNG -- per question, one anchor per stratum.

\paragraph{Distance stratum schemes.}
\begin{itemize}
  \item \textbf{absolute}: fixed boundaries $[0,1], [2,5], [6,30], [31,300], [301,\infty)$. Narrow-GT dataset (TallyQA).
  \item \textbf{relative}: upper bound $\mathrm{hi} = \max(\text{absolute\_floor}, \text{fraction} \cdot |gt|)$ with fractions $[10\%, 30\%, 100\%, 300\%, \infty]$. Wide-GT datasets.
  \item \textbf{relative\_s1}: first stratum only ($|a - gt| \leq \max(1, 0.10 \cdot gt)$).
\end{itemize}

\section{Cross-architecture verification: $\gamma$-$\beta$ residual-stream bridge (Qwen3-VL)}
\label{app:gamma-beta}

\todoNote{\S5.3 routing-and-integration framework 의 partial cross-architecture verification. Main panel 밖 architecture (Qwen3-VL) 위 \textit{방향성 prediction 만} 검증.}

\paragraph{Setup.} For Qwen3-VL, take the layer-wise residual-stream difference $\gamma - \beta$ between Thinking ($\gamma$) and Non-thinking ($\beta$) modes on identical inputs. Extract the SVD K=1 subspace of this difference and apply it as a layer-projection. Measure anchoring change.

\paragraph{Result.}
\begin{itemize}
  \item Late layers (L=29--34) $\rightarrow$ anchoring \emph{decreases} (positive).
  \item Mid layer (L=20) $\rightarrow$ anchoring \emph{increases} (negative -- sign reversal).
\end{itemize}

\paragraph{Framework prediction agreement.} The sign-reversal pattern matches the framework's layer-routing directionality (late = integration, mid = routing) $\rightarrow$ partial prospective verification.

\paragraph{Caveats.}
(i) $N=1$ architecture (Qwen3-VL only).
(ii) K=1 only -- the original K=8 quantitative ratio ($\times 12.7$) is \emph{falsified}.
(iii) Not strictly prospective (framework already existed when tested) -- only \emph{directional} cross-architecture verification.
